# 🏥 Hospital Project

## 📌 Project Overview
This project focuses on analyzing hospital operational data to generate meaningful business insights and improve healthcare decision-making.

The analysis includes:
- Patient records
- Treatments
- Lab results
- Pharmacy orders
- Follow-ups
- Hospital information

The project demonstrates skills in:
- Data Cleaning
- Data Analysis
- SQL
- Python
- Power BI Dashboarding
- Business Insights
  ---

# 📂 Dataset Files

- patients_raw.csv
- treatments_raw.csv
- lab_results_raw.csv
- pharmacy_orders_raw.csv
- followups_raw.csv
- hospitals.csv
